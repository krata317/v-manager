# V-MANAGER: Student Study Planner

## Overview

V-MANAGER is a simple web-based planner for students to manage tasks, subjects, events, and study resources.

## Features

* User login and registration
* Add and manage tasks and subjects
* Filter tasks by subject
* Monthly calendar view
* Add and track events
* Embedded study resources

## Technologies Used

* HTML
* CSS
* JavaScript
* LocalStorage

## How to Run

1. Open `index.html` in a browser
2. Register and login
3. Start using the planner

## Note

All data is stored in the browser using localStorage and is not shared across devices.
